题目背景
SYY 又在班里大喊大叫。由此可见，SYY 在班级里很不受欢迎。

题目描述
清明节，SYY 决定不能再这么下去，他决定列一个名单，把班级里 n 个同学对他的好感度从高到低排个序，以此来看他可以让每位同学帮多少忙（已知帮一次忙要消耗 k 点好感度，不足 k 的部分不算）。

输入格式
先输入两个整数 n, k（1≤n<10^5,1≤k≤10^4）。
接下来一行 n 个正整数，表示每个同学对他的好感度（[0,100]之间）。

输出格式
n 行，输出好感度最大的同学可以帮他的次数
注意：0次请输出N0。
输入输出样例
输入 #1
10 5
12 0 0 0 5 7 8 15 20 0
输出 #1
4
输入 #2
5 3
0 0 0 0 0
输出 #2
N0
